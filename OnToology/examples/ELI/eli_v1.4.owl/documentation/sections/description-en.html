ELI Metadata Ontology: Description back to ToC

An ontology to describe legislation, based on the FRBR model, that encodes the ELI metadata fields. See the classes "LegalResource", "LegalExpression" and "Format" as entry points. Interested readers can find more information about ELI on the ELI register at http://eurlex.europa.eu/eli and on the Office of Publications vocabularies website at https://op.europa.eu/en/web/eu-vocabularies/eli.
